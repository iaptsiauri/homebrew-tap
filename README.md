# Homebrew Tap for pit

## Install

```bash
brew install iaptsiauri/tap/pit
```

## Usage

```bash
cd your-repo
pit              # launch TUI dashboard
pit new my-task  # create a task
pit run my-task  # run in background
pit status       # see all tasks
```

## Update

```bash
brew upgrade pit
```
